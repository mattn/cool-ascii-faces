# cool-ascii-faces for golang

golang implementation of [cool-ascii-faces](https://github.com/maxogden/cool-ascii-faces)

## Usage

```go
for c := range faces.Stream() {
	fmt.Println(c)
}
```

## Installation

```
$ go get github.com/mattn/cool-ascii-faces
```
